
# Weather Web Application UI

A responsive weather web application interface built with HTML and CSS. This project demonstrates modern CSS techniques including Flexbox and Grid for creating a clean, user-friendly weather dashboard.

## 📋 Project Overview

This project focuses on designing a weather web application UI that displays comprehensive weather information including current conditions, hourly forecasts, and daily predictions. The design emphasizes responsive layout principles and semantic HTML structure.

## ✨ Features

- Current Weather Section
  
- Location details with local time

- Weather condition with icon placeholder

- Temperature display (°C & °F)

- Comprehensive weather metrics:

- "Feels like" temperature

- Wind speed (KPH & MPH)

- Humidity percentage

- Cloud cover

- UV index

- Pressure

- Visibility

- Wind direction

- Dew point

- Heat index

- Wind chill

- Last updated timestamp

## Hourly Forecast

- Scrollable horizontal cards

- Time-based forecasting (e.g., 1 PM, 2 PM)

- Weather icons

- Short condition descriptions

- Temperature readings

## Daily Forecast

- 3-5 day forecast cards

- Maximum and minimum temperatures

- Weather icons

- Condition descriptions

## 🛠️ Technologies Used

- Tailwindcss
